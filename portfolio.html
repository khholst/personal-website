Philadelphia Public Transportation App

An application that visualises Philapelphia's public transportation data in real time, including on route buses, trolleys, trains and bus stops/train stations. It's possible to select specific vehicles to access more data such as:

  • destination
  • next stop
  • current stop
  • timing
  • free seats

As a basemap the application uses Mapbox tiles. The data that is being processed and visualised is fetched from the SEPTA API service every 10 seconds.

Explore the application or take a look at the demo video.

Tech Stack:

  • Javascript
  • p5.js
  • mappa.js
  • SEPTA API

United States Crime Map

An interactive web application to depict different crimes in the United States from 1960 to 2014. The buttons are for selecting the crime type, which is shown on the map and the graph. Using the slider it's possible to select a specific year to see the result on the map and by clicking on a state on the map, its data throughout the years is drawn on the graph. Also, when the selected year is not the last year in the dataset, it's possible to play a timelapse and see the chages in the selected crime type throughout the years without changing the slider manually.

As a basemap the app uses CARTO tile services. The crime dataset used can be browsed here.

Explore the application.

Tech Stack:

  • Javascript
  • p5.js
  • mappa.js

A Quiz Game of Estonian Cities

A quiz game where the goal is to mark Estonian cities as close and as fast as possible to their location on the map. In the start menu the palyer has to enter a name and the score is later added to the database and top 5 scores are shown on the leaderbard. For one guess the time limit is 5 seconds and after guessing, correct and guessed locations are shown on the map. In addition, the submenu shows the distance, time and score gained. The game ends when all of the 45 cities have been guessed.

The basemap is created with QGIS software, using data from Estonian Land Board tile services.

Play the game.

Tech Stack:

  • Node.js
  • JavaScript
  • MongoDB Atlas
  • p5.js
  • QGIS

Night Length Application

An application that calculates night length on any given day in any given location on Earth. In addition, it finds the time of sunrise, sunset and the times of civil, nautical and astronomical twilights. The calculations are based on a formula from "Almanac For Computers".
